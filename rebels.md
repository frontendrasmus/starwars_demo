# Rebel Alliance — Field Handbook (CONFIDENTIAL)

_Classification: SAPPHIRE-9 · Eyes only · Compiled by Alliance Intelligence (A-INT), 0 BBY_

This handbook documents active and dormant Alliance facilities, key field
agents, code phrases, and standard operating procedure for cell leaders.
Memorise. Do not transcribe. If captured, swallow the mnemonic shard.

---

## Active Bases

### Echo Base — Hoth, Anoat sector
Code name: **TAUNTAUN**. A buried-ice command fortress on the sixth planet
of the Hoth system. Officially decommissioned after the Imperial assault on
3 ABY, but rumours persist of a small caretaker garrison maintaining the
shield generator. Approach vector: Rendili → Vergesso → blind jump to grid
H-4429. Daily countersign rotates at 0600 GST.

### Yavin 4 — Massassi Outpost
Code name: **GREEN PYRAMID**. The original headquarters of the Rebellion
during the Battle of Yavin. Concealed inside a Massassi temple complex on
the fourth moon of Yavin Prime. The temple's stonework masks ion emissions;
sensors register only ancient wildlife. Primary hangar holds 24 X-wings and
a complement of Y-wing bombers. Operational since 2 BBY.

### Crait — outer rim mineral mine
Code name: **SALT FOX**. An abandoned First Order–era mine repurposed by
the Resistance, but originally an Alliance fallback site. Salt flats above
crystalline beds hide a blast door designed by Mon Calamari engineers.
Approachable only by ski-speeder; surface is too reflective for orbital
imaging.

### Dantooine — the old base
Code name: **AURORA**. The Alliance's first formal base, evacuated before
the destruction of Alderaan. Now a deception site — a skeleton crew runs
fake transmissions to draw Imperial attention away from active cells.
Anyone arriving without the phrase "the moisture is fair this season" is
to be detained.

### Kothlis — Bothan listening post
Code name: **WHISPER-ROOST**. A signal-intelligence facility hidden inside
a freight processing station on Kothlis. Run jointly with Bothan Spynet
operatives. Specialises in decrypting HoloNet courier traffic. Authorised
visitors only — Bothans take questions of identity very seriously.

### Polis Massa — medical asylum
Code name: **WHITE STONE**. A Kallidahin research outpost converted into a
field hospital and forensics lab. Site of the recovery of Padmé Amidala
(19 BBY, classified). Currently runs tissue regeneration trials on
casualties from the Mid Rim front.

### Cyrkon — Outer Rim safehouse
Code name: **DUSTSWELL**. A cantina-and-warehouse compound in the
backstreets of Motok, Cyrkon's largest spaceport city. Operated under
cover by a Sullustan family. Used to extract burned agents and run them
through a four-stage identity rebuild.

## Standing Cells

- **Phoenix Cell** — Lothal sector. Mobile fleet built around the Ghost.
  Lead: GHOST-1. Specialises in raids on Imperial supply convoys.
- **Massassi Group** — Yavin sector. Front-line strike force.
  Lead: DODONNA. Includes Red, Gold, Blue, and Green squadrons.
- **Crimson Eel** — Corellian sector. Smuggler-affiliated cell. Moves
  contraband, fuel, and refugees through Corellian Run hyperlanes.
- **Iron Squadron** — Mykapo. Volunteer pilots flying salvaged Y-wings.
  Officially folded into Massassi after the loss of Mykapo, 2 BBY.
- **Spectres** — Lothal. The original Phoenix progenitor cell. Six members.

## Recognition Protocols

Counter-phrases rotate every fourteen days. The current set (period
SAPPHIRE-9) is:

1. _Challenge_: "Have you tasted the wine of Naboo?"
   _Response_: "Only the vintage from the Lake Country."
2. _Challenge_: "Is the moon over Yavin?"
   _Response_: "Always — and so is the temple."
3. _Challenge_: "How fares the moisture this season?"
   _Response_: "Fair, when the binary suns are kind."

If an agent fails the response by more than one word, treat as compromised
and disengage. If they fail by exact one word, they may be using the
fallback **TWILIGHT** sequence — proceed cautiously.

## Standard Operating Procedure — burned agent extraction

1. Confirm burn signal (the agent's chrono will broadcast a 14-second
   pulse on band 47.3 once every six minutes).
2. Route the agent to the nearest **DUSTSWELL** safehouse — never directly
   to an active base.
3. Identity rebuild: new biometric tag, fresh chain code, four-stage cover
   story drilled over no fewer than nine standard days.
4. Reassignment review by Alliance Intelligence. Agents may not return to
   their previous cell for at least one calendar year.

## Known compromised assets

- **WAVECREST** (Garel) — believed compromised by ISB after the courier
  Fenn Rau was tracked back to it. Do not transmit to this site.
- **SUNSPOT** (Atollon) — destroyed by the Empire 2 BBY in a coordinated
  strike led by Grand Admiral Thrawn. All hands lost except the Phoenix
  Cell evacuees.
- **ASHFALL** (Mustafar perimeter) — never operational; the deep-recon
  team inserted there in 4 BBY did not return. Site classified as a black
  hole — all approach routes are now Imperial honey traps.
